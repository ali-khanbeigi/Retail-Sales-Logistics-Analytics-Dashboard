# Retail Sales & Logistics Analytics Dashboard

## Overview
This project is an end-to-end data analytics dashboard focused on retail sales performance and logistics operations. It provides insights into key business metrics such as sales trends, shipment distribution, delivery delays, and carrier performance.

## Objectives
- Analyze retail sales performance over time
- Evaluate logistics efficiency and delivery performance
- Identify delays and bottlenecks in shipment processes
- Provide data-driven insights for better decision-making

## Key Metrics
- Total Sales & Revenue Trends
- Shipment Weight Distribution
- Order Fulfillment Rate
- Delivery Delay Analysis
- Country/Customer-based shipment insights

## Tools & Technologies
- Power BI / Excel (Dashboard Visualization)
- SQL Server

## Key Insights
- Revenue is concentrated in a few top categories, especially Beverages and Dairy.
- Performance depends heavily on a small set of customers and sales reps.
- The 2008 decline requires further investigation.
- Overall fulfillment is strong, with 98.46% shipped orders, but 236 total delay days indicate room for improvement.
- Shipper GVSUA has the highest delay days (97), while ZHISN is the most reliable (50 delays, 99.27% shipped).
- Shipment weight is concentrated in USA and Germany.
- Shipment volume mirrors sales, peaking in 2007 and declining in 2008.
